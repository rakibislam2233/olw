## 📖 About The Project

Welcome to the **Google Property Management Suite**. Built from the ground up for modern property managers, this platform effortlessly centralizes your workflow. From orchestrating complex booking schedules and dispatching cleaning crews to maintaining precise financial oversight, our SaaS provides an uncompromising, premium toolset to scale your hospitality business.

We have prioritized a flawless user experience, backed by cutting-edge technology, ensuring real-time syncing and high reliability for everyday operations.

---

## ✨ Core Features

*   **🏠 Seamless Property Setup:** Rapidly onboard new properties with rich descriptive data, high-quality images, and dynamic pricing rules.
*   **📅 Unified Booking Calendar:** Automatically prevent double-bookings and view all upcoming reservations across multiple properties in a single pane of glass.
*   **🧹 Automated Cleaning Schedules:** Dispatch cleaning workflows instantly upon guest checkout to maintain impeccable hygiene standards.
*   **🕵️ Real-time Guest Monitoring:** Track check-ins, check-outs, and active stays to maintain security and ensure smooth transitions.
*   **💰 Automated Expense Tracking:** Log maintenance costs, operational expenses, and overhead seamlessly. Generate robust financial reports instantly.
*   **🤝 Instant Guest Support Channel:** Provide world-class hospitality via centralized communication hubs handling all guest inquiries directly inside the app.

## 🚀 Getting Started

Follow these steps to set up the development environment on your local machine using **pnpm**.

### Prerequisites

Ensure you have Node.js and `pnpm` installed.

```bash
# Install pnpm globally if you haven't already
npm install -g pnpm
```

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/rakibislam2233/olw.git
    cd olw
    ```

2.  **Install dependencies:**
    ```bash
    pnpm install
    ```

3.  **Start the development server:**
    ```bash
    pnpm dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to view the application.
